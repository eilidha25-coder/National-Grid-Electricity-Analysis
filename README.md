# UK Electricity Demand and Renewable Energy Analysis using Python
Exploring UK electricity demand using National Grid data and Meteostat weather observations. Analysis includes seasonal demand trends, temperature relationships and embedded renewable generation using Python.

## Data Sources

- National Grid ESO Open Data Portal – Half-hourly GB electricity demand data (2025).
- Meteostat – Daily weather observations for Glasgow (2025).

- ## Requirements

- Python 3.x
- pandas
- NumPy
- matplotlib
- SciPy
